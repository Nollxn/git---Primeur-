# 🏢 Projet Aïchi

Aïchi est une application de gestion des commandes développée en **C# avec Windows Forms**, suivant une architecture **MVC (Modèle - Vue - Contrôleur)** et connectée à une **base de données SQL**.  
Ce projet a été réalisé en groupe dans le cadre du **BTS SIO (Services Informatiques aux Organisations)**.

---

## 🚀 Objectif du projet

L’objectif du projet Aïchi est de fournir une solution simple et intuitive permettant à une entreprise de gérer efficacement :
- Ses **clients**
- Ses **articles**
- Ses **commandes**

L’application permet d’effectuer toutes les opérations essentielles de gestion.
---

## ⚙️ Technologies utilisées

- **Langage :** C#
- **Interface :** Windows Forms
- **Architecture :** MVC
- **Base de données :** SQL Server / MySQL
- **IDE :** intelliJ
